# charles-keygen

An open-source Charles Proxy keygen via RC5 cracking. Works on all version, even on newest v5 releases.

![whoops](view.png)

## Usage

```sh
# git clone git@github.com:duraki/charles-keygen.git
# ...
$ javac CharlesKeygen.java && java CharlesKeygen "0xduraki"

# * GENERATED LICENSE:
#  [0xduraki:936FB63A91BBEFFD08]
```
